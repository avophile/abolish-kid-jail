# Contributing to Abolish Kid Jail

Thank you for your interest in helping parents protect their children with disabilities from discriminatory school discipline! This project aims to be accessible and welcoming to contributors of all backgrounds.

## How to Contribute

### For Parents and Advocates

Your lived experience is invaluable:
- Share your story (anonymously if preferred) about navigating manifestation determination hearings
- Suggest improvements to the letter templates based on what worked for you
- Identify gaps in the educational resources
- Test the application and provide feedback on usability
- Suggest state-specific guidance that would be helpful

### For Special Education Attorneys and Advocates

Your legal expertise helps ensure accuracy:
- Review letter templates for legal accuracy
- Provide guidance on IDEA regulations and implementation
- Suggest key language that strengthens parent demands
- Identify common school district tactics and how to counter them
- Review state-specific requirements

### For Developers

Help build the technical foundation:
- Review and improve code quality
- Add new features from the roadmap
- Improve accessibility and mobile responsiveness
- Optimize PDF generation
- Add automated tests
- Improve documentation

### For Designers

Make the tool more accessible and user-friendly:
- Improve the user interface and experience
- Ensure accessibility for users with disabilities
- Design clear, easy-to-understand forms
- Create visual aids to explain manifestation determination rights
- Design printable resources

## Getting Started

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature-name`)
3. Make your changes
4. Test thoroughly
5. Commit your changes with clear, descriptive commit messages
6. Push to your fork
7. Create a pull request

## Code of Conduct

This project is about protecting vulnerable children and supporting their parents. All contributions should:

- Be respectful and supportive
- Center the needs of parents and students
- Prioritize accessibility
- Use clear, plain language (avoid unnecessary jargon)
- Respect privacy and confidentiality

## What We're Looking For

### High Priority
- Letter template development and refinement
- Simple, accessible web interface
- PDF generation functionality
- Educational content about manifestation determination rights
- Documentation and user guides

### Medium Priority
- State-specific guidance
- Multiple language support
- Mobile optimization
- Printable resources and checklists

### Future Ideas
- Integration with document signing services
- Tracking tools for parents managing multiple hearings
- Community forum for parent support
- Database of school district responses (anonymized)
